# Tableau-Project

**Tableau Sales Performance Dashboard**

An interactive Tableau dashboard project designed to analyze sales performance, profit margins, return rates, customer segments, and product categories using data visualization techniques. The dashboard provides business insights through charts, maps, KPIs, and trend analysis to support data-driven decision-making.

### Features

* Sales analysis by category and sub-category
* Profit margin and return rate tracking
* Regional sales visualization using maps
* Customer segment performance analysis
* Top 5 product insights
* Interactive and user-friendly Tableau dashboard

### Tools & Technologies

* Tableau
* Data Visualization
* Business Analytics

### Project Objective

The main objective of this project is to transform raw business data into meaningful visual insights that help understand sales trends, profitability, and overall business performance.
